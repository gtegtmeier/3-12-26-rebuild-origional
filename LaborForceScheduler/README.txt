LaborForceScheduler Runtime Quick Start
=======================================

1) Run the program directly
- Double-click: Run_LaborForceScheduler.bat
  (or run scheduler_app_v3_final.py with Python)

2) Create a desktop icon
- Double-click: Install_Desktop_Icon.bat
- This creates a desktop shortcut named: LaborForceScheduler.lnk

3) Branding image location
- assets/petroserve.png

4) Python requirement
- Python is required to run this project from source.
- If needed in the future, this can be packaged as a standalone EXE.
